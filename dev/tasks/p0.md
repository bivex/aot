# Task P0: English Subject-Predicate Detection

## Status: Working (Morphology + Grammar + Frequency-based Disambiguation)

The English syntactic module has a complete pipeline: UniMorph-based morphology with closed-class verb paradigms, a 61-rule GLR grammar, frequency-based homonym disambiguation from UD-EWT, and heuristic SP detection.

### Changes Implemented:

1.  **Morphology (UniMorph + Closed Classes)**:
    - Integrated UniMorph English data (~540K lemmas) via `unimorph_to_aot_v2.py`.
    - Injected closed-class suppletive verb paradigms: **BE** (is/am/are/was/were/been/being), **HAVE** (have/has/had/having), **DO** (do/does/did/done/doing) — tagged as VBE.
    - Modal verbs (can/must/will/shall/may/ought) tagged as MOD.
    - Particles (not, never) injected as PART.
    - 17 POS tags: NOUN, ADJ, VERB, VBE, MOD, NUMERAL, CONJ, INT, PREP, PART, ARTICLE, ADV, PN, ORDNUM, PRON, POSS, PN_ADJ.

2.  **Grammar (`simple_syn.grm` — 61 rules)**:
    - NP: bare nouns, pronouns, proper nouns; with articles, adjectives, possessives, numerals.
    - VP: simple verbs, copula (VBE+ADJ/NP/PP), progressive (is running), passive (was broken), perfect (has eaten), perfect progressive (has been running), passive progressive (is being eaten).
    - Modal: can/must/will + verb, + progressive, + passive.
    - Negation: VBE/VOD + not + verb/adj/ing/pp.
    - Infinitive with "to": is to go.
    - Adverbial modifiers in VP.
    - Coordination: NP and NP, VP and VP, verb and verb.
    - Subordinate clauses: SP + CONJ + SP, CONJ + NP + VP.
    - Root: SYN_GROUP covers NP, VP, PP, ADVERB, PART, SP.

3.  **Frequency-Based Homonym Disambiguation**:
    - Generated `StatData.txt` and `WordData.txt` from UD-EWT corpus (201K tokens, 20K+ word forms).
    - Compiled into binary weight files (`*homoweight.bin`, `*wordweight.bin`) via StatDatBin and word_freq_bin.
    - Implemented `DeleteHomOneToThousand()` for English (10x threshold) — eliminates rare homonyms before syntax analysis.
    - POS mapping from UD-EWT to AOT: personal pronouns → PN, AUX → VBE/MOD, DET → ARTICLE, etc.
    - Patched StatDatBin to skip unknown words instead of throwing exceptions.

4.  **Subject-Predicate Heuristics** (`CEngSentence::BuildSubjAndPredRelation`):
    - Locates the first verb-like unit as **Predicate**, closest preceding noun-like unit as **Subject**.
    - Assigns `isSubj: true` and `m_iPredk` for visualization.

5.  **Stability Fixes**:
    - Fixed off-by-one in `CoordConjTypesCount` (21→20).
    - Separated coordinating vs subordinating conjunctions in `EngOborDic`.

### Current Limitations:

- **Morph dictionary gaps**: RUNS, WALKING, RUNNING have no VERB forms in the morph dictionary (UniMorph only has NOUN). Need to add regular verb inflections.
- **Low-frequency ambiguities**: BOOKS (NOUN/VERB ratio 9:1) survives the 10x threshold — borderline cases remain.
- **Stub methods**: `GleicheSubjectPredicate`, `is_infinitive`, `is_morph_participle` in `CEngGramTab` always return false.
- **CAN as VERB**: In some contexts (e.g., "Birds can fly"), CAN is tagged VERB instead of MOD — the MOD/VERB ratio may not be high enough in the weight data.

## Next Steps:
- Add regular verb inflections to UniMorph data (3rd person -s, -ing, -ed, -en forms).
- Implement `GleicheSubjectPredicate` for subject-predicate agreement.
- Tune the disambiguation threshold or use per-POS thresholds.
