# Поддерживаемые части предложения (Syntactic Relations)

В этом документе перечислены синтаксические отношения и части предложения, которые парсеры системы AOT умеют выделять для различных языков.

## 1. Русский язык (RusSynan)
Модуль обладает наиболее полной поддержкой и использует комбинацию GLR-грамматик и сложных эвристик на C++.

*   **Подлежащее (SUBJ)** — тег `isSubj`.
*   **Сказуемое (PREDK)** — тег `isPredk`.
*   **Прямое дополнение** (вин. падеж без предлога).
*   **Косвенное дополнение** (падежные формы, предложные группы).
*   **Определения (ATTR)** — согласованные прилагательные, причастия.
*   **Обстоятельства** (наречия, предложные группы).
*   **Вводные слова** (через специальный словарь и эвристики).
*   **Однородные члены** (через правила сходства).

## 2. Немецкий язык (GerSynan)
Высокий уровень поддержки на основе GLR-грамматик, разработанных в рамках проекта DWDS.

*   **Подлежащее (Subject)**.
*   **Сказуемое (Predicate)**.
*   **Объекты (Direct/Indirect Objects)**.
*   **Обстоятельства**.
*   **Придаточные предложения** (Clause borders).

## 3. Английский язык (EngSynan)
В настоящее время находится в стадии прототипа (Heuristic-based).

*   **Подлежащее и Сказуемое (Subject-Predicate)** — **NEW!** Реализовано через эвристику: первый глагол в клаузе считается сказуемым, а ближайшее существительное/местоимение слева — подлежащим.
*   **Именные группы [NP]** (Noun Phrases).
*   **Глагольные группы [VP]** (Verb Phrases).
*   **Предложные группы [PP]** (Prepositional Phrases).
*   **Наречные группы**.

## 4. Украинский язык
На данный момент поддержка синтаксиса минимальна. Основной фокус на морфологии (VESUM).

---

### Вывод в API (JSON)
Синтаксические связи передаются в массиве `groups` внутри каждого варианта разбора предложения:
- `isSubj: true` — помечает связь между подлежащим и сказуемым.
- `start` / `last` — индексы слов в предложении, между которыми установлена связь.
