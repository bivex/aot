# Ukrainian Morphological Dictionary

This directory will contain compiled binary dictionary files for Ukrainian after building.

The following files are generated by the build process:
- morph.bases - base forms and paradigms
- morph.forms_autom - morphological automaton
- morph.annot - annotations
- gramtab.json - grammar table (copied from source)
- morph.options - options
- chomoweight.bin, lhomoweight.bin, fhomoweight.bin - homonym weight models
- cwordweight.bin, lwordweight.bin, fwordweight.bin - word weight models
- npredict.bin - prediction model

## Requirements
Actual data files must be present in `Source/morph_dict/data/Ukrainian/` before building.
See `Source/morph_dict/data/Ukrainian/README.md` for data preparation instructions.
